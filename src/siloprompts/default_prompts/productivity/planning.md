# Productivity & Planning Prompts

## Project Planning

```
Help me plan the following project:

**Project Name:** [NAME]
**Goal:** [WHAT YOU WANT TO ACHIEVE]
**Deadline:** [TIMELINE]
**Resources:** [TEAM/BUDGET/TOOLS]

**Requirements:**
- Break down into phases/milestones
- Task list with dependencies
- Risk identification
- Resource allocation
- Timeline estimation

**Deliverables:**
[LIST EXPECTED OUTPUTS]
```

## Task Prioritization

```
Help me prioritize these tasks:

**Tasks:**
1. [TASK 1] - [BRIEF DESCRIPTION]
2. [TASK 2] - [BRIEF DESCRIPTION]
3. [TASK 3] - [BRIEF DESCRIPTION]
[ADD MORE]

**Context:**
- Goals: [CURRENT GOALS]
- Deadline: [IF ANY]
- Constraints: [LIMITATIONS]

**Use Eisenhower Matrix or similar framework to:**
- Categorize by urgency/importance
- Suggest order of execution
- Identify tasks to delegate/eliminate
```

## Meeting Agenda Creation

```
Create an agenda for a [TYPE] meeting about:

**Topic:** [MEETING SUBJECT]
**Duration:** [TIME ALLOCATED]
**Attendees:** [WHO WILL ATTEND]
**Objective:** [WHAT NEEDS TO BE ACCOMPLISHED]

**Include:**
- Time-boxed agenda items
- Discussion topics
- Decision points
- Action items template
- Pre-meeting preparation needed
```

## Learning Plan

```
Create a learning plan for:

**Skill/Topic:** [WHAT YOU WANT TO LEARN]
**Current Level:** [BEGINNER/INTERMEDIATE/ADVANCED]
**Time Commitment:** [HOURS PER WEEK]
**Timeline:** [WEEKS/MONTHS]
**Goal:** [DESIRED OUTCOME]

**Provide:**
- Structured curriculum
- Resource recommendations (books/courses/projects)
- Milestones and checkpoints
- Practice exercises
- Assessment methods
```

## Decision Framework

```
Help me make a decision about:

**Decision:** [WHAT YOU'RE DECIDING]

**Options:**
1. [OPTION 1]
2. [OPTION 2]
3. [OPTION 3]

**Important Factors:**
- [FACTOR 1]
- [FACTOR 2]
- [FACTOR 3]

**Constraints:**
[LIST CONSTRAINTS]

**Provide:**
- Decision matrix
- Weighted scoring
- Risk analysis for each option
- Recommendation with reasoning
```

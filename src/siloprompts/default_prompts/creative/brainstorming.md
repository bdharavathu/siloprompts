# Creative & Brainstorming Prompts

## Idea Generation

```
Generate [NUMBER] creative ideas for:

**Topic/Challenge:** [DESCRIBE]
**Target Audience:** [WHO IS THIS FOR]
**Constraints:** [BUDGET/TIME/RESOURCES]

**Requirements:**
- Innovative and unique approaches
- Practical feasibility
- Brief explanation for each idea
- Potential impact

**Focus Areas:**
- [AREA 1]
- [AREA 2]
```

## Problem-Solving Brainstorm

```
Help me brainstorm solutions for:

**Problem:** [DESCRIBE THE PROBLEM]
**Context:** [BACKGROUND INFORMATION]
**Constraints:** [LIMITATIONS]

**Use creative thinking techniques:**
- Lateral thinking
- SCAMPER method
- First principles thinking

**Provide:**
- 10+ diverse solution approaches
- Pros and cons for top 3 ideas
- Implementation considerations
```

## Naming/Branding Ideas

```
Generate names for [PRODUCT/COMPANY/PROJECT]:

**Description:** [WHAT IT IS/DOES]
**Target Market:** [WHO WILL USE IT]
**Brand Personality:** [ADJECTIVES]
**Keywords:** [RELEVANT WORDS]

**Requirements:**
- Memorable and unique
- Easy to pronounce
- Available domain name (if applicable)
- No negative connotations
- Provide 15-20 options with brief rationale
```

## Story/Scenario Creation

```
Create a [TYPE] story/scenario about:

**Theme:** [MAIN THEME]
**Setting:** [WHERE/WHEN]
**Characters:** [WHO IS INVOLVED]
**Objective:** [PURPOSE OF STORY]

**Elements to Include:**
- [ELEMENT 1]
- [ELEMENT 2]

**Tone:** [Serious/Humorous/Inspirational/etc]
**Length:** [APPROXIMATE WORDS]
```

## Analogy and Metaphor Generation

```
Create analogies/metaphors to explain:

**Concept:** [WHAT YOU'RE EXPLAINING]
**Target Audience:** [WHO NEEDS TO UNDERSTAND]
**Complexity:** [SIMPLE/MODERATE/COMPLEX]

**Provide:**
- 5-7 different analogies
- Explanations showing the parallel
- Best use cases for each analogy
```

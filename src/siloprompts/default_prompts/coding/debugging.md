# Debugging Prompts

## General Bug Diagnosis

```
I'm encountering an issue with the following code. Please help me debug:

**Problem Description:**
[DESCRIBE THE ISSUE]

**Expected Behavior:**
[WHAT SHOULD HAPPEN]

**Actual Behavior:**
[WHAT IS HAPPENING]

**Code:**
[PASTE YOUR CODE]

**Error Message (if any):**
[PASTE ERROR MESSAGE]
```

## Error Message Analysis

```
I'm getting the following error. Please explain what it means and how to fix it:

**Error:**
[PASTE ERROR MESSAGE]

**Context:**
[BRIEF DESCRIPTION OF WHAT YOU WERE DOING]

[RELEVANT CODE]
```

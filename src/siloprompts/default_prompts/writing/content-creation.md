# Content Creation Prompts

## Blog Post Writing

```
Write a blog post about [TOPIC]:

**Target Audience:** [DESCRIBE AUDIENCE]
**Tone:** [Professional/Casual/Technical/etc]
**Word Count:** [APPROXIMATE LENGTH]

**Key Points to Cover:**
- [POINT 1]
- [POINT 2]
- [POINT 3]

**Include:**
- Engaging introduction
- Clear section headings
- Practical examples
- Compelling conclusion
- SEO-friendly structure
```

## Technical Documentation

```
Create technical documentation for [FEATURE/API/TOOL]:

**Purpose:** [WHAT IT DOES]
**Target Users:** [DEVELOPERS/END-USERS/etc]

**Sections Needed:**
- Overview and introduction
- Prerequisites
- Installation/Setup instructions
- Usage examples
- API reference (if applicable)
- Troubleshooting
- FAQ

**Style:** Clear, concise, with code examples
```

## Email/Message Templates

```
Draft a [TYPE] email for:

**Purpose:** [WHY YOU'RE WRITING]
**Recipient:** [WHO YOU'RE WRITING TO]
**Tone:** [Formal/Informal/Friendly/etc]
**Key Message:** [MAIN POINT]

**Include:**
- Professional greeting
- Clear subject line
- Concise body
- Call to action (if needed)
- Appropriate closing
```
